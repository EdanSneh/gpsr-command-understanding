Rephrase a command to a robot

Given a command that someone gave to a household helper robot, rephrase it in your own words. Don't try to be overly formal, simply write what you would say if you wanted the robot to accomplish the same command.

Do:

  • Capture all of the same information as the original command
  • Write something different than the original command
  • Use natural english
  • Consider varying sentence structure and using different words that mean roughly the same thing
  • Keep instructions in the same order, if there are multiple instructions in one command

Original Command

Pick up the apple from the table and place it on the counter

Rephrased

Move the apple on the table to the counter please

Original Command

Take Fred from the bedroom to the kitchen

Rephrased

Get Fred from the bedroom to the kitchen

Original Command

Go to the foyer, meet Alice, and lead her to the living room.

Rephrased

Guide Alice to the living room.

Reason: The rephrased command doesn't include all of the information from the original command. Specifically, it doesn't include that Alice can be found in the foyer.

Original Command

Find a person in the bedroom and tell them the time.

Rephrased

Tell the time then find someone in the bedroom.

Reason: The rephrased command changes the meaning of the command. Specifically, it tells the robot to tell the user the time, when it should tell the robot to tell someone in the bedroom the time.
